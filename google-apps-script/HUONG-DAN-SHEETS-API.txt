CÁCH MỚI (KHUYÊN DÙNG) — Google Sheets API, không cần Apps Script
================================================================

Cách này ghi thẳng vào Sheet từ Vercel, tránh lỗi redirect / POST.

Bước 1 — Google Cloud
  - Vào https://console.cloud.google.com
  - Tạo project (hoặc chọn project có sẵn).
  - Menu "APIs & Services" → "Library" → tìm "Google Sheets API" → Enable.

Bước 2 — Service account + key JSON
  - "APIs & Services" → "Credentials" → "Create credentials" → "Service account".
  - Đặt tên bất kỳ → Create → Done (bỏ qua optional grants).
  - Vào service account vừa tạo → tab "Keys" → "Add key" → "Create new key" → JSON → tải file về máy (ví dụ key.json).

Bước 3 — Chia sẻ Google Sheet
  - Mở file Google Sheet chứa dữ liệu.
  - Nút Share (Chia sẻ) → dán EMAIL trong file JSON, trường "client_email"
    (dạng xxx@xxx.iam.gserviceaccount.com).
  - Quyền: Editor (Người chỉnh sửa) → Send.

Bước 4 — Base64 cho Vercel (Linux / Mac trong terminal):
  base64 -w0 key.json
  (Mac: base64 -i key.json | tr -d '\n')
  Copy TOÀN BỘ chuỗi base64 (một dòng).

Bước 5 — Biến môi trường Vercel
  Project → Settings → Environment Variables (Production):

  GOOGLE_SHEET_ID
    = ID trang tính: trong URL
      https://docs.google.com/spreadsheets/d/XXXXXXXX/edit
      chỉ lấy phần XXXXXXXX.

  GOOGLE_SERVICE_ACCOUNT_B64
    = chuỗi base64 vừa copy (một dòng).

  XÓA hoặc để trống GOOGLE_SCRIPT_URL nếu trước đó bạn dùng Apps Script
  (để tránh nhầm).

  Save → Deployments → Redeploy.

Bước 6 — Cài dependency local (để Vercel build đúng)
  Trong thư mục project, chạy:  npm install
  Commit file package.json và package-lock.json rồi push lên Git (nếu deploy từ Git).

Bước 7 — Thử
  - Mở site đã deploy, hoàn thành bước 7, gửi khảo sát.
  - Mở Sheet: phải có dòng mới (cột A = thời gian, B = participant id, C = JSON).

Lỗi thường gặp:
  - 403 / Permission denied: chưa Share Sheet cho đúng client_email (Editor).
  - 502: kiểm tra GOOGLE_SHEET_ID đúng; base64 không bị cắt dòng khi dán vào Vercel.
  - API chưa bật: bật Google Sheets API trong Google Cloud.
